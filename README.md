# Weather Analysis

A Python project analysing 1957 Met Office MIDAS weather observations.

## Purpose

This project uses real weather data to investigate temperature patterns at Portland Bill Lighthouse and produce statistics drawn from the data through code.

## Questions answered

- What was the hottest day?
- What was the coldest day?
- What are the average temperatures by month?
- Which month has the greatest temperature range?

## Tools used

- Python
- Pandas
- Matplotlib

## Data source

Met Office MIDAS Open UK Land Surface Stations Data.

## Results

The program identifies temperature extremes, calculates monthly averages, and creates graphs showing weather patterns.
